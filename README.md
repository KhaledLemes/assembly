# Assembly

I created this repo to store my learning in progress in the learning of Assembly.<br>
<br>

## Purpose
The reason I am learning Assembly, is not only to better understand how computers and memory actually work, but also to improve the necessary knowledge to work with reverse engineering and malware analysis.

## Architecture and tools
Everything here is written using x86\_64 architecture.<br>
All code here is written using [robalb/x86-64-playground](https://github.com/robalb/x86-64-playground) with GNU Assembly.
